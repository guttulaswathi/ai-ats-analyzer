# AI ATS Analyzer

An AI-powered Applicant Tracking System (ATS) that compares resumes with job descriptions and provides:
- Match percentage
- Matching & missing skills
- Strengths
- Improvement suggestions

## Tech Stack
- Python (Flask)
- Google Gemini API
- HTML, Tailwind CSS, JavaScript

## How to Run
```bash
pip install -r requirements.txt
python main.py
